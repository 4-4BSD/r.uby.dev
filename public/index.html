Matz

Matz is a statically linked, 3MB binary that teaches you Ruby by answering questions from official documentation, ri, rdoc, and the mruby source code. He grounds every answer in primary sources.

Matz is also available through a publicly accessible sshd 🙂
So you can try Matz without installing anything, and it is free to use within reason. 

ssh matz@r.uby.dev

Documentation

Documentation from the ri tool is available through an HTTP+JSON interface, intended for scripts, terminals, AI agents, and any other software that speaks HTTP. 

##
# List ri entries
fetch -qo - 'https://r.uby.dev/ri/index'

##
# Render an ri entry
fetch -qo - 'https://r.uby.dev/ri/search?q=Enumerable'

Indexed sources:

Open Source

All r.uby.dev services are 100% open source.

The documentation search can be self-hosted and for FreeBSD users there's a make install target alongside an rc.d script. Matz is also open source and can be built from source.
