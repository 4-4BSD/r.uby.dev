Matz

Matz is a statically linked, 3MB binary focused on mruby. He answers questions from mruby documentation, the mruby source tree, mrbgems, and related primary sources.

Try Matz via SSH. Free to use. Nothing to download or install. 

$ ssh matz@r.uby.dev

Documentation

mruby guides, source code, and known mrbgem manifests are available over HTTP+JSON for scripts, terminals, AI agents, and any other software that speaks HTTP.

Endpoint Description
GET /api/guides/index List mruby guide documents.
GET /api/guides/search?q=QUERY Search mruby guides.
GET /api/guides/read?q=PATH Read an mruby guide document.
GET /api/mruby/index List mruby source files.
GET /api/mruby/search?q=QUERY Search mruby source code.
GET /api/mruby/read?q=PATH Read an mruby source file.
GET /api/mrbgems/index List known mrbgem manifests.
GET /api/mrbgems/search?q=QUERY Search known mrbgem manifest filenames.
GET /api/mrbgems/read?q=GEM Read an mrbgem manifest.

Current sources:

  • mruby guides
  • mruby source code
  • mrbgem manifests

Open Source

All r.uby.dev services are 100% open source.

The mruby service can be self-hosted and for FreeBSD users there's a make install target alongside an rc.d script. Matz is also open source and can be built from source.
