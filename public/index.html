4.4BSD

A free-to-use service for the BSD community that provides full-text search across FreeBSD's official documentation and handbooks. The HTTP interface is intended for scripts, terminals, and tools that speak JSON.

Books

The following books have been indexed.
More could be added in the future.

  • FreeBSD User's Handbook
  • FreeBSD Developer's Handbook
  • FreeBSD Porter's Handbook

How can I use it?

The primary 4.4bsd.dev interface is HTTP, and the intended audience is scripts, terminals, AI agents, and any other piece of software that can speak HTTP+JSON. It is probably not very useful for human eyes. :) 

##
# Search the user's handbook
$ fetch -qo - 'https://4.4bsd.dev/handbook/u/search?q=pkg'

##
# Search the developer's handbook
$ fetch -qo - 'https://4.4bsd.dev/handbook/d/search?q=compiler'

##
# Search the porter's handbook
$ fetch -qo - 'https://4.4bsd.dev/handbook/p/search?q=Makefile'

Where is it used?

Robert is a statically linked, 3MB binary designed to teach and troubleshoot FreeBSD. Robert uses this service for full-text search, and it helps ground Robert's logic and answers in official FreeBSD documentation.

For now that's the only project I know of. This service is still young and relatively unknown. If your project uses this service, and you'd like to have it featured on this page, please reach out by email - 0x1eef@hardenedbsd.org

Is it open source?

Absolutely. This service is 100% open source, and it can be self-hosted. It is optimized to run on FreeBSD, and includes a make install target alongside an rc.d script for managing the web server as a service.

4.4BSD

The name 4.4BSD might look strange to an unfamiliar eye but it carries a lot of weight – both historically and for me personally.

History

It stands as the final release of BSD from the Computer Systems Research Group at Berkeley. The final release served as the foundation for what went on to become FreeBSD, OpenBSD, and NetBSD.

Personal

I was 14 or 15 years old when I was sent a CD of FreeBSD 4.4-RELEASE. That's a long time ago. Windows 98 can be your anchor. I picked it up at the post office, and after playing with it for the summer, I had become obsessed with it. It felt like a system that wanted to teach you how it works, and I loved that. FreeBSD 4.4-RELEASE became my introduction to BSD, and the name 4.4BSD comes from the final release from CSRG at Berkeley.

Thank you

Thank you to everyone who has contributed to FreeBSD, and its documentation. The documentation is seriously impressive, and it covers far more than I realized. There isn't one handbook, there are multiple, and I haven't even indexed all of them.
