Ruby.dev

Ruby documentation services for the Ruby community. Publicly accessible, no signup required.

Matz

Matz is a statically linked, 3MB binary that teaches you Ruby by answering questions from official documentation, ri, rdoc, and the mruby source code. He grounds every answer in primary sources.

Matz is also available through a publicly accessible SSH endpoint 🙂 You can try Matz without installing anything, and it is free to use within reason. 

ssh matz@r.uby.dev

No signup. No account. Just SSH.

Documentation Search

Full-text search across Ruby's documentation via an HTTP+JSON interface, intended for scripts, terminals, AI agents, and any other software that speaks HTTP. 

##
# Search Ruby documentation
fetch -qo - 'https://r.uby.dev/ri/search?q=Enumerable'

##
# Search rdoc
fetch -qo - 'https://r.uby.dev/rdoc/search?q=Array#map'

Open Source

Both services are 100% open source. The search engine can be self-hosted. Matz is also open source and can be built from source.
