Matz

Matz is designed to help you understand mruby using guide documents, source code, mrbgem manifests, and related primary project material.

How can I try Matz?

SSH

The quick way is SSH. No install required. It is free to use within reason. If you're not sure what Matz is, this is a good way to find out without installing anything.

$ ssh matz@r.uby.dev
HTTP

The same mruby reference material is also exposed through JSON endpoints for scripts, terminals, and agents.

##
# List mruby guides
$ fetch -qo - 'https://r.uby.dev/api/guides/index'

##
# Search mruby source
$ fetch -qo - 'https://r.uby.dev/api/mruby/search?q=mrb_gc'

What makes Matz different?

  • Publically accessible via SSH
    Matz is available for anyone to use, for free, over SSH
  • Focused on mruby.
    Matz treats mruby guides, source code, and mrbgems as first-class reference material.
  • Grounded answers.
    Answers are expected to trace back to primary project sources rather than generic Ruby summaries.
  • Terminal-native.
    You can use Matz over SSH, and the backing reference endpoints are plain HTTP+JSON.
  • Read-only by design.
    The public tools inspect documentation and source material. They do not modify project files.
  • Small deployment model.
    Matz follows the same statically linked, low-friction shape as the other r.uby.dev services.

Abilities

Matz tools are read-only. The first implemented tool is guide-index; the remaining tools map one-to-one with the public endpoints and follow the same shape.

Name Purpose Endpoint Status
guide-index List mruby guide documents. GET /api/guides/index Implemented
guide-search Search mruby guide documents. GET /api/guides/search?q=QUERY Planned
guide-read Read an mruby guide document. GET /api/guides/read?q=PATH Planned
mruby-index List files in the mruby source tree. GET /api/mruby/index Planned
mruby-search Search mruby source code. GET /api/mruby/search?q=QUERY Planned
mruby-read Read an mruby source file. GET /api/mruby/read?q=PATH Planned
mrbgem-index List known mrbgem manifests. GET /api/mrbgems/index Planned
mrbgem-search Search known mrbgem manifest filenames. GET /api/mrbgems/search?q=QUERY Planned
mrbgem-read Read an mrbgem manifest. GET /api/mrbgems/read?q=GEM Planned
