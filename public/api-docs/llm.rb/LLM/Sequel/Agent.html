Overview

Sequel plugin for persisting LLM::Agent state.

This wrapper reuses the same record-backed runtime surface as Plugin, but builds an LLM::Agent instead of an LLM::Context. Agent defaults such as model, tools, schema, instructions, and concurrency are configured on an internal agent subclass.

def self.configure(model, options = EMPTY_HASH, &block)
  options = DEFAULTS.merge(options)
  model.db.extension :pg_json if %i[json jsonb].include?(options[:format])
  model.instance_variable_set(:@llm_agent_options, options.freeze)
  block_given? ? model.instance_exec(model.agent, &block) : nil
end
