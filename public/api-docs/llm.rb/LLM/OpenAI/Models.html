Overview

The LLM::OpenAI::Models class provides a model object for interacting with OpenAI's models API. The models API allows a client to query OpenAI for a list of models that are available for use with the OpenAI API.

Examples:

#!/usr/bin/env ruby
require "llm"

llm = LLM.openai(key: ENV["KEY"])
res = llm.models.all
res.each do |model|
  print "id: ", model.id, "\n"
end

def all(**params)
  query = URI.encode_www_form(params)
  req = LLM::Transport::Request.get(path("/models?#{query}"), headers)
  res, span, tracer = execute(request: req, operation: "request")
  res = ResponseAdapter.adapt(res, type: :models)
  tracer.on_request_finish(operation: "request", res:, span:)
  res
end
