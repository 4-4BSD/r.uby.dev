Overview

The ZAI class implements a provider for zAI.

Examples:

#!/usr/bin/env ruby
require "llm"

llm = LLM.zai(key: ENV["KEY"])
ctx = LLM::Context.new(llm, stream: $stdout)
ctx.talk "Hello"
