Overview

Manages an Async::Reactor on a background thread. Work is submitted through a thread-safe queue and run inside the reactor. The reactor and its fibers stay on one thread.

def stop
  @inbox << :stop
  @thread.join(5)
  @thread.kill if @thread.alive?
end
